# 🍕 Pizza Hut Sales Intelligence Case Study  

> 🚀 Leveraging SQL to transform transactional data into strategic business insights  

---

## 🌟 Overview  
Executed a **comprehensive SQL-based sales intelligence analysis** on Pizza Hut data to evaluate revenue performance, customer behavior, product efficiency, and time-driven sales patterns.

This project demonstrates the ability to move beyond querying and **apply SQL as a tool for business problem-solving and decision support.**

---

## 🎯 Business Objective  
To uncover key revenue drivers, optimize product performance, and analyze customer purchasing behavior by transforming raw sales data into meaningful, decision-ready insights.

---

## 🧠 Analytical Approach  

- 🔍 Structured data exploration across multiple relational tables  
- 📊 Revenue decomposition and order-level analysis  
- 📈 Time-based trend identification (hourly, daily, monthly)  
- 🛒 Product and category performance evaluation  
- 👥 Customer purchasing behavior analysis  
- ⚙️ Application of advanced SQL techniques for deeper insights  

---

## 🔍 Key Insights  

- 💰 A small group of pizza types contributes a disproportionate share of total revenue  
- 📊 Large-sized pizzas dominate overall sales volume, indicating strong customer preference  
- ⏰ Peak order periods reveal clear demand concentration during specific hours  
- 📅 Sales performance varies significantly by day, highlighting operational opportunities  
- 🛒 Customers tend to purchase multiple items per order, increasing average order value  
- 📈 Category-level analysis identifies high-performing segments and optimization gaps  

---

## 🛠️ SQL Expertise Demonstrated  

- ✔ Complex multi-table joins for relational data analysis  
- ✔ Advanced aggregations (SUM, AVG, COUNT) for KPI extraction  
- ✔ Subqueries for layered analytical logic  
- ✔ Window functions (RANK, running totals) for advanced insights  
- ✔ Creation of reusable views for reporting efficiency  
- ✔ Structured query design for scalable data analysis
